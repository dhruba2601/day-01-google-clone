# 🔍 Google Homepage Clone

A pixel-close clone of Google's homepage built with pure HTML and CSS.
No JavaScript. No frameworks. Just the fundamentals.

Built as part of my Full Stack Developer journey in 2026.

---

## 📸 Preview

![Google Clone Preview](preview.png)

---

## ✅ Features

- Colorful Google logo with exact brand colors
- Rounded search bar with shadow effect
- "Google Search" and "I'm Feeling Lucky" buttons
- Fully centered layout using CSS Flexbox
- Clean white background matching real Google

---

## 🛠️ Built With

| Technology | Purpose |
|-----------|---------|
| HTML | Structure — logo, input, buttons |
| CSS | Styling — colors, layout, flexbox |

## 🧠 What I Learned

- HTML page structure and semantic tags
- CSS Flexbox for centering layouts
- How to use span tags to style individual characters
- CSS box-shadow for realistic input styling
- border-radius for rounded pill shapes
- letter-spacing for tight logo typography
- Google's exact brand color codes

## 🚀 How To Run

1. Download or clone this repository
2. Open google.html in any browser
3. See the Google homepage clone

No installation needed. No dependencies. Just open and use.
